## 第一章
程序为0和1组成的位。
8个位组成一起为字节。
每个字节表示程序中的某些字符。

只由 `ASCII` 字符构成的文件为——文本文件
其他文件为———二进制文件

编译一个可执行的 `hello world`  需要4个步骤:
hello.c -> 预处理器(cpp) -> hello.i -> 编译器(ccl) -> hello.s -> 汇编器(as) -> hello.o -> 链接器(ld) -> hello可执行文件
